# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| latest (`main`) | ✅ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them via email to **ryanjbush@gmail.com** with the subject line `[SECURITY] Meridian AI — <brief description>`.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested remediation if known

You can expect an acknowledgement within **48 hours** and a resolution timeline within **7 days** for critical issues.

## RBAC & Auth

This platform implements role-based access (Admin, Analyst, Reviewer, Viewer). If you discover an authentication bypass or privilege escalation vulnerability, treat this as **critical** and disclose immediately.

## Scope

This project is a portfolio/demonstration platform. All transaction data is fully synthetic. It is not connected to any real payment network or financial institution.
